# PSAJ Informatika XII — Kurikulum Merdeka

Website ujian online untuk Penilaian Sumatif Akhir Jenjang (PSAJ) mata pelajaran Informatika Kelas XII.

## Deploy ke Vercel

1. Buat akun di [vercel.com](https://vercel.com)
2. Install Vercel CLI: `npm i -g vercel`
3. Jalankan: `vercel --prod` di folder ini
4. Atau drag & drop folder ini ke [vercel.com/new](https://vercel.com/new)

## Fitur

- 43 soal dari 6 materi Informatika
- 6 bentuk soal: PG, PGK, Menjodohkan, Benar/Salah, Isian, Uraian
- Timer 120 menit otomatis
- Setelah selesai, siswa TIDAK melihat nilai/jawaban
- Dashboard Guru (PIN: 1234) untuk melihat semua jawaban
- Penilaian uraian manual oleh guru
- Export nilai ke CSV
